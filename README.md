# Super_Store--Sales-Dashboard
SuperStore Sales Dashboard and Forecasting

Overview:
This project presents an end-to-end Power BI dashboard developed using SuperStore sales data. 
It provides insights into sales, profit, order quantity, delivery performance, customer segments, and geographic patterns.
The dashboard includes a 15-day forecasting model and demonstrates strong analytical, visualization, and storytelling skills.

Dashboard Features:
- Sales and profit performance across categories and subcategories
- Segment analysis such as consumer, corporate, home office
- Shipping mode performance with delivery delays and efficiency patterns
- Geographic performance insights at regional and state levels
- Time intelligence for YoY and MoM trend analysis
- 15-day sales forecasting integrated into the dashboard
- Interactive region-wise slicers for Central, East, South, and West
- Professionally designed dark theme for better readability

Tools and Technologies:
Power BI Desktop
Power Query for cleaning and shaping data
DAX for time intelligence, calculations, and custom measures
Excel/CSV dataset for input

Repository Structure:
The repository includes:
- A .pbix Power BI report file
- A Data folder containing the dataset
- An Images folder containing dashboard previews
- A README describing the project


Sample DAX Measures:
Total Sales = SUM(Superstore[Sales])
Total Profit = SUM(Superstore[Profit])

Sales Forecasting uses Power BI's built-in analytics model.

How to View the Dashboard:
1. Download the .pbix file
2. Open it using Power BI Desktop
3. Load or refresh the dataset if needed
4. Use slicers to explore regional and category-based trends

Skills Demonstrated:
- Report creation with professional UI/UX
- Time intelligence using DAX
- Forecasting using built-in analytics
- Regional performance comparison
- Data cleaning and transformation in Power Query
